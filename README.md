# Таблица с JSON данными. Редактирование, валидация данных, сортировка, поиск
<b>Технологии:</b> <i>Vue js, vue-CLI, Vuetify, moment js.</i>

Клиентское web-приложение, отображающее таблицу с данными из JSON-файла. 

## Project setup
```
npm install
```
